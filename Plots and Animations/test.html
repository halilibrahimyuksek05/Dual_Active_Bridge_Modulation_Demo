Basic Operating Modes

A

Mode A: Positive power flow with positive inductor current

B_1

Mode B: Reversing polarity from positive to negative, positive current flow

B_2

Mode B: Reversing polarity from positive to negative, negative current flow

C

Mode C: Positive power flow with negative inductor current

D_1

Mode D: Reversing polarity from negative to positive, negative current flow

D_2

Mode D: Reversing polarity from negative to positive, positive current flow
